Supabase setup required

Replace the two placeholders in the script with your Supabase credentials, then run the SQL below to create the songs table.

SQL to run in Supabase SQL Editor:

create table songs (id bigint generated always as identity primary key, title text, artist text, youtube_id text, audio text, theme text, difficulty text, text text, grammar text, vocab jsonb, created_at timestamptz default now());

The user_progress table is the same as the course app — reuse it if you already have it, or create a new one per the comments in the script.
